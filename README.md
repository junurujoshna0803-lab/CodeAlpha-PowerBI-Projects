## 👥 1. HR Analytics Dashboard

### 📌 Overview

An interactive dashboard designed to analyze employee data and provide insights into workforce distribution and demographics.

### ✨ Features

- Total Employees KPI
- Average Salary KPI
- Employees by Department
- Gender Distribution
- Experience Distribution
- Department Filter
- Gender Filter

### 🛠 Tools Used

- Microsoft Power BI
- Microsoft Excel
